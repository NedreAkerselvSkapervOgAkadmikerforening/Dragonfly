# Dragonfly
Dragonfly er NASAs nyeste rakketsatsning, med mål om å skape en modulær,  vedvarende raketmodell.

Dragonfly kan overordnet sett inndeles i fire hovedsystemer:

1. **Dragonfly**: Selve rakettkroppen. Alt fra strukturen til recovery-systemet og motoren.
2. **Nebula**: Flydatamaskinen, PCB og annen elektronikk i raketten.
3. **Atlas**: Bakkestasjonen, tett knyttet til Nebula, og består av en bærbar datamaskin som kjører et dashboard, kontrollpanel og en mikrokontroller med radio for kommunikasjon.
4. **Forge**: Launch rail og systemer rundt som raketten starter på under oppskyting. Også ansvarlig for antennelse og sikkerhetssystemer.
